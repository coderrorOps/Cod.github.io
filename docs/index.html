HTML

    Уровни

  • Легко
  • Средне
  • Сложно

HTML по своей сути не язык программирования. Это язык разметки гипертекстовых документов. Иными словами он отвечает за расположение в документе ваших текстов, рисунков, таблиц и т.д. В этом языке нет логических функций и поэтому в нем нельзя решать математические задачи, но их можно красочно описывать. Язык HTML был создан для написания сайтов и в настоящее время он является самым основным для их создания.

CSS

    Уровни

  • Легко
  • Средне
  • Сложно

CSS — это язык описания внешнего вида документа, написанного с помощью HTML. Целью создания CSS было разделение описания логической структуры документа. Сразвитием технологий люди старались сделать сайты более функциональными и яркими, но все параметры внешнего вида представлялись в html тегах. Поэтому cегодня CSS используется практически на всех сайтах.

Pascal

    Уровни

  • Легко
  • Средне
  • Сложно

Pascal — один из наиболее известных языков программирования, используется для обучения программированию в старших классах и на первых курсах вузов, является основой для ряда других языков. Pascal был представлен в качестве небольшого и эффективного языка, способствующего хорошему стилю программирования, использующему структурное программирование и структурированные данные.

Python

    Уровни

  • Легко
  • Средне
  • Сложно

Python — высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен. В то же время стандартная библиотека включает большой объём полезных функций. Примерно раз в 2 года выходят обновления. Сегодня это самый быстрорастущий язык программирования.
